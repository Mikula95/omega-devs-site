# Omega-Devs - Company Website

A professional website for Omega-Devs, a software development company specializing in innovative solutions.

## 🚀 Features

- **Modern Design**: Clean, professional design with blue and black color scheme
- **Responsive Layout**: Fully responsive design that works on all devices
- **Interactive Components**: Smooth animations and hover effects
- **Company Showcase**: Complete sections for services, projects, and company information
- **Contact Form**: Interactive contact form for client inquiries

## 🛠️ Built With

- **Vue.js 3** - Progressive JavaScript framework
- **Vite** - Fast build tool and development server
- **CSS3** - Modern styling with gradients and animations
- **JavaScript ES6+** - Modern JavaScript features

## 🚀 Getting Started

### Prerequisites

- Node.js 20.19+ or 22.12+ (currently works with 20.18.0 with warnings)
- npm or yarn package manager

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd omega-devs-site
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and visit `http://localhost:5173` (or the port shown in terminal)

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## 🔧 Customization

To customize the website:

1. **Colors**: Update the color variables in the CSS files
2. **Content**: Modify the text content in each Vue component
3. **Projects**: Update the projects array in `Projects.vue`
4. **Contact Info**: Update contact details in `Contact.vue` and `Footer.vue`
5. **Services**: Modify service offerings in `Services.vue`
